This Java program can be used to:
a) Establish connection to a service that provides timely AQ data for selected urban areas in Finland (Enfuser API) 
b) Create AQ queries and recieve the requested data (both point queries and raster queries are supported).
c) Show raster queries (pollutant concentrations in an area) on interactive OpenStreetMap
d) Download longer AQ timeseries on CSV or JSON to local drive
